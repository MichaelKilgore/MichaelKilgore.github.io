  • RobinhoodAPIFun

    • This program when run will pull data from a specified stock, and create its own graph, then develop trend lines, then finally using advanced trend line trading strategies attempts to predict what the future prices of a stock will be, and turn a profit.

      In order to create this program I had to integrate multiple APIs. First I scraped a website for names of all the stocks that are open to the public. Then I used the Robinhood API to pull stock history and form a graph. Then I used matplotlib to plot the data pulled, after that I created trend lines based on the data. After that based on the trend lines present, using advanced trend line trading strategies the program will buy and sell accordingly.
